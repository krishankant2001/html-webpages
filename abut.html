About HTML

THESE ARE IMPORTANT TAGS IN HTML :

  • <!DOCTYPE>
  • <html>
  • <head>
  • <title>
  • <body>
  • <h1>to<h6>
  • <p>
  • <br>
  • <a>
  • <img>
  • <iframe>
  • <form>
  • <fieldset>
  • <legend>
  • <button>
  • <textarea>
  • <ul>
  • <ol>
  • <li>
  • <dl>
  • <style>

<DOCTYPE> :

This tag defines the type of document.

<html> :

this tag defines an html document and it is used in starting of documents and ended at end of the document.

<head> :

this tag defines information about the document and internel CSS is used in this tag.

<title> :

defines a title for document and it is shown on top of the window.

<body> :

this tag defines the body of the document.

<h1>to<h6> :

defines html headings

<p> :

this tag is used to write the paragraph

<br> :

this tag is used for line breaking.

<a> :

used for adding links in page.

<img> :

use for adding image in html page.

<iframe> :

use for adding inline frame in page.

<form> :

used to create forms

<fieldset> :

used to create border lines around a form content.

<legend> :

used for heading of form.

<button> :

this tag is used for creating different types of buttons.

<textarea> :

used for putting additional space for text.

<ul> :

used to create unorderd list.

<ol> :

used to create ordered list

<li> :

used to create list items.

<dl> :

used to create description lists.

<style> :

used to put styles in pages with CSS
